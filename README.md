# Task Nest

A lightweight task management app with drag-and-drop task organization, filtering, local storage persistence, due dates, and labels.

## Features

- Drag and drop tasks between columns
- Add, edit, and delete tasks
- Save data with `localStorage`
- Filter tasks by priority and status
- Due dates and labels for each task

## Usage

1. Open `index.html` in your browser.
2. Click **New Task** to create a task.
3. Drag tasks between **To Do**, **In Progress**, and **Done**.
4. Use the filter dropdowns to narrow tasks by status or priority.

## Notes

Tasks are stored locally in your browser and will remain after reloads on the same device/browser.
